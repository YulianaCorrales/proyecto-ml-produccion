# Proyecto IA listo para producción

## Fase 1 - Modelo predictivo

### Dataset
Titanic - Machine Learning from Disaster (Kaggle)

### Archivos generados
- `fase-1/modelo.ipynb`: Notebook con entrenamiento y predicción
- `fase-1/submission.csv`: Predicciones finales sobre el dataset test
- `fase-1/modelo_titanic.pkl`: Modelo entrenado guardado para futuras fases

### Cómo ejecutar
1. Descargar los datos desde Kaggle.
2. Guardar los archivos `train.csv`, `test.csv` y `gender_submission.csv` en `fase-1/data/`.
3. Abrir el notebook `fase-1/modelo.ipynb`.
4. Ejecutar todas las celdas en orden.
5. El notebook generará `submission.csv` y `modelo_titanic.pkl`.
